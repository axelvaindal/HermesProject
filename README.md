# Hermes Project

## ABOUT

Hermes Project is an autonomous connected device in order to store and analyze data during shipping.
This project is able to handle various form of data, including:

- Temperature
- Humidity
- GPS
- Acceleration and Angular Acceleration
- Angular Position
- DateTime
- Light level
